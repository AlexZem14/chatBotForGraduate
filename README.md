# Чат бот для абитуриентов ФЭиВТ ВолгГТУ
Данный чат-бот предназначен для абитуриентов факультета электроники и вычислительной техники ВолгГТУ. В нем имеется два режима - информационный и интерактивный. Первый режим содержит информацию об университете и направлениях, имеющихся на факультете. Также с помощью него можно узнать сведения касаемо поступления. Второй режим представляет собой диалог с пользователем и предоставляет информацию по выбору направления.
Ссылка на группу ВКонтакте, в которой находится чат-бот: https://vk.com/chatBotForGraduate
